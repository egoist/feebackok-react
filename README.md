# Feedback OK React

React component for [Feedback OK](https://feedbackok.com).

## Install

```bash
yarn add feedbackok-react
```

## Usage

```tsx
import { FeedbackForm } from 'feedbackok-react'

const App = () => {
  return <FeedbackForm config={{ pid: 'xxx' }} />
}
```
